# CEYLON JOURNEY — 의사결정 이력 (DECISIONS.md)

> 최종 수정: 2026-04-29

---

## DEC-001 | 2026-04-29 | 브랜드 분리
- **결정:** '여행능력자들' 브랜드로 판매하지 않고, CEYLON JOURNEY라는 새 브랜드로 운영
- **이유:** 스리랑카 프리미엄 패키지는 기존 호텔 추천 채널과 성격이 다름. 독립 브랜드로 분리하여 고급 이미지 유지
- **영향:** consumer_site.html에서 여행능력자들 관련 모든 내용 삭제, CEYLON JOURNEY 브랜딩 적용

## DEC-002 | 2026-04-29 | 상품 페이지 디자인 방향
- **결정:** 블로그 형태가 아닌 프리미엄 여행사 수준의 상품 상세 페이지로 구축
- **이유:** 기존 페이지가 블로그 느낌으로 구매 전환이 어려움. 하나투어/모두투어급 상품 상세 페이지로 변경
- **영향:** 풀스크린 히어로, 장소별 이미지+매력 설명, 7일 전일정 상세 일정표 등 전면 재설계

## DEC-003 | 2026-04-29 | TW Standard System v1 적용
- **결정:** TW B2B에서 검증된 5개 모듈 표준 시스템을 CEYLON JOURNEY에도 적용
- **이유:** 프로젝트 간 작업 방식 통일, 작업 이력 추적, 대표님 검수 편의
- **영향:** Business Docs / Page Gallery / 검증 워크플로 / 작업 기록 / 자율 작업 큐 5개 모듈 추가

## DEC-004 | 2026-04-29 | 정식 도메인 추후 연결
- **결정:** 현재는 Vercel 자동 도메인(ceylon-journey-three.vercel.app)으로 운영, 정식 도메인은 추후 연결
- **이유:** 상품/브랜드가 먼저 확정된 후 도메인 결정
- **영향:** 없음 (도메인 변경 시 Vercel 설정만 추가하면 됨)
