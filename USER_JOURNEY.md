# CEYLON JOURNEY — 사용자 여정 (USER_JOURNEY.md)

> 최종 수정: 2026-04-29

---

## 소비자 여정

```
[유입] → [상품 탐색] → [상세 확인] → [예약 문의] → [결제] → [출발 준비] → [여행] → [후기]
```

### 1단계: 유입
- 경로: 유튜브 / 카카오톡 / SNS / 검색엔진 / 지인 추천
- 랜딩: consumer_site.html (메인 상품 페이지)

### 2단계: 상품 탐색
- 히어로 영역에서 핵심 정보 확인 (6박7일, 5성급, 3식, 20인)
- 스리랑카 매력 포인트 확인 (시기리아, 나인아치, 갈레)

### 3단계: 상세 확인
- 7일 전일정 상세 일정 확인 (장소별 이미지+설명)
- 호텔 정보 확인
- 요금/포함·불포함 확인

### 4단계: 예약 문의
- 카카오톡 채널로 문의
- 출발 확정 일정, 정확한 요금, 싱글룸 추가 등 상담

### 5단계: 결제
- [추후 확정] 계좌이체 / 카드결제

### 6단계: 출발 준비
- 관리자가 admin_final.html에서 예약 관리
- 여권 수집 → 비자 신청 → 항공 발권 → 최종 안내서 발송

### 7단계: 여행
- 한국어 가이드 동행
- 전일정 5성급 호텔

### 8단계: 후기
- [추후 확정] 후기 수집 → 사이트 게시

---

## 관리자 여정

```
[예약 접수] → [여권 수집] → [비자 신청] → [항공 발권] → [안내서 발송] → [출발]
```

- 모든 단계를 admin_final.html에서 관리
- 예약 상태: 입금대기 → 예약금완료 → 예약확정 → 잔금필요 → 잔금완납 (또는 취소)
